<Steps>
<Step title="Install KiloCode">
  ```bash
  npm install -g kilocode
  ```
</Step>

<Step title="Start Vibe Kanban">
  Once KiloCode is installed, launch Vibe Kanban:

  ```bash
  npx vibe-kanban
  ```

  You can now select KiloCode when creating task attempts.
</Step>
</Steps>
